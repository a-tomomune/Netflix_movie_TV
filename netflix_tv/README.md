# 1. 目的
 Netflix作品データを分析し、作品の特徴や配信傾向を可視化するとともに、テキスト分析やレコメンドシステムを通じて、データ分析・機械学習の手法を実践的に学ぶことを目的とする。
 
# 2. 使用データ
　Kaggle：[Netflix Movies and TV Shows](https://www.kaggle.com/datasets/shivamb/netflix-shows/data)

# 3. 分析方針
本プロジェクトでは、Netflix作品データに対して、**前処理** → **SQLによる集計分析** → **Pythonによる可視化・詳細分析** → **レコメンドシステム構築**の流れで分析を行う。

### 1. データ前処理（Python）
- 欠損値・重複データの確認と処理
- データ型の変換（日付・年など）

### 2. 基礎分析（SQL）
SQLを用いてデータ全体の傾向を把握する。
- 映画・TV Showの件数
- 各特徴量別ランキング

### 3. 可視化・詳細分析（Python）
SQLで集計した結果を可視化し、作品の特徴を分析する。
- データ分析
- 時系列分析
- 文章分析
  - 感情分析
  - TF-IDF分析

### 4. レコメンドシステム
作品概要やジャンルなどのテキスト情報を特徴量として利用し、推薦システムを構築する。

# 5. 主な結果
### 1. Netflix作品全体の特徴
- 全8,807作品のうち、**映画が約7割（69.6%）**、TVが約3割（30.4%）を占める。
- 配信作品の多くは**成人向け（TV-MA・TV-14）** であり、全年齢向け作品は比較的少ない。
- TVは**1シーズン完結**の作品が最も多い。

### 2. 配信傾向
- Netflixへの作品追加は**2019年がピーク**となり、それまでは右肩上がりに増加。
- 公開年は**2018年制作作品**が最も多く、公開から約１年で配信される作品が多い傾向が見られる。
- 映画は2016~2019年に急増し、TVは2020年まで緩やかに増加。

### 3. 国別の特徴
- **アメリカ**の作品数（2818件）が圧倒的に多く、全体の約３割を占める。
- **日本**はアニメ作品の存在感が大きい。

### 4. ジャンル・年齢制限
- 映画では**劇映画**や**インターナショナル映画**が中心で、ドキュメンタリーやスタンダップコメディも多い。
- TVでは**キッズ番組**や**国際番組**が上位を占めた。
- 映画・TVともに成人向け作品が中心となっている。
- TVの方が映画より子供向け作品の割合が大きい。

### 5. キャスト・監督
- 最も多く作品を出している監督は**Rajiv Chilaka**。
- 最も多く出演しているキャストは**Anupam Kher**。
- 日本作品では**Takahiro Sakurai**、**Yuki Kaji**など声優同士の共演が多く確認された。

### 6. 時系列分析
- 2019年にかけて作品数を増やしてきたが、それ以降は追加数が減少傾向にある。
- TVは映画よりも最新作品が多い傾向にある。
- TVでは**7歳以上の子ども向け（TV-Y7）** 、**全年齢対象（TV-G）** の追加作品に増加傾向が見られた。

### 7. 文章分析（作品説明文）
- 説明文の平均の長さは約143文字で、大きなばらつきは見られなかった。
- 頻出単語は**life、young、family、world**など、人生を表す単語が中心。
- 日本作品では**high、school**など、学校を表す単語が多かった。
- 感情分析では全体的に見ると、中立～ややポジティブな傾向が多い。


# 6. ビジネス示唆
- Netflixはファミリー層よりも、10代後半～成人向けコンテンツを中心に差別化している可能性がある。
- 比較的新しい作品を継続的に追加することで、サブスクリプション継続率向上を戦略的に取っていると考えられる。
- 各国のローカル作品を充実させることで、各国のユーザー確保と新規市場の開拓につなげていると考えられる。

# 7. 工夫した点
- SQLで基礎的な分析をすることで、大まかな情報を持ってPythonで分析することができた。
- derector、castなどのカンマ区切りのデータを分割し、一人ずつ分析できるようにした。
- 制作年と追加年を分析することで、どのような作品が追加されているのか傾向をみた。
- 映画とTVを分けて分析することで、それぞれの特徴を明確にした。
- 分析のみで終わらせず、推薦システムという日常へ応用できるものを作成した。

# 8. 分析の限界
- 顧客層が分からないため、どんな作品にどんな需要があるのかわからない。
- 視聴データがないため、需要や人気を直接知ることができない。
- オリジナル作品の区別がないため、Netflix制作の戦略分析までできない。
- どのくらい配信されているのかわからない。
